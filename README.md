# LeetCode Solutions

A collection of my solutions to LeetCode problems.

## About

This repository contains my personal solutions to various LeetCode problems as I work through them. Each problem is organized in its own directory with the solution code and problem description.

## Structure

Each problem folder follows this format:
- **Solution file**: Contains the implementation
- **README**: Problem description, examples from LeetCode, and notes for myself about the solution

## Languages

Solutions are primarily written in Python.

## Purpose

This repository serves as a personal archive of my problem-solving journey and coding practice. Feel free to browse through the solutions!

---

*Note: These solutions reflect my approach to solving these problems and may not always be the most optimal solution.*
